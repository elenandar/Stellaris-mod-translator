# Stellaris Mod Translator

Локальная настольная платформа для безопасного перевода модификаций Stellaris на русский язык.

Пользователь выбирает папку с модами, запускает анализ и получает отдельный русский мод-компаньон для каждого исходного мода. Исходники никогда не изменяются. Повторный запуск переводит только новые и изменённые единицы, а технически небезопасный результат не публикуется.

## Статус

Проект находится на этапе архитектурного решения. Реализация ещё не начата. Этот репозиторий фиксирует предлагаемую стратегию, технологический стек, обязательные правила и план работ, чтобы решение о разработке принималось до написания продукта.

## Контракт продукта

- локальное desktop-приложение, в первую очередь для macOS, с готовностью к Windows и Linux;
- режим «выбрать папку → перевести всё», без обязательной ручной работы;
- один управляемый русификатор-компаньон на один исходный мод;
- оригинальные моды и Workshop-каталоги доступны только для чтения;
- модель получает только переводимый текст, безопасные атомы и минимальный контекст, но не редактируемый код;
- неизвестный синтаксис изолируется, а не угадывается;
- техническая целостность проверяется до публикации; литературность и соответствие лору — отдельными проверками;
- локальный Ollama — первый провайдер; облачные провайдеры подключаются явно и только с согласия пользователя;
- все задания возобновляемы, публикация атомарна, последняя исправная версия доступна для отката.

## Следующая точка решения

Если концепция принята, разработка начинается не с интерфейса и не с вызова LLM, а с исследовательского корпуса и доказательства безопасного, обратимого разбора локализации. Переход к автоматическому переводу разрешён только после прохождения технического шлюза M2 из [дорожной карты](docs/roadmap.md).
